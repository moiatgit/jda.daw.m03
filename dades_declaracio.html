Declaració de variables¶

La declaració de variables en Python és diferent de la d’altres llenguatges (com ara Java)

A Python, declararem una variable en assignar-li un valor a un nom. Per exemple:

edat = 19

D’aquesta manera, automàticament tenim una nova variable edat que conté el valor 19.

Inicialització: Quan assignem un valor inicial a una variable es diu que l’estem inicialitzant.

A Python la declaració requereix inicialització.

Quan inicialitzem una variable en Python, l’intèrpret decideix el tipus de la variable a partir del valor que li assignem. En el cas de l’exemple edat = 19, com que el tipus del literal és enter, el tipus de la variable edat passa a ser enter.

També podem saber el tipus d’una variable en Python fent servir type():

>>> edat = 19
>>> type(edat)
<class 'int'>

Més endavant li podem assignar un valor d’un altre tipus i Python l’acceptarà sense problemes. Seguint l’exemple anterior:

>>> type(edat)
<class 'int'>
>>> edat = 'dinou'
>>> type(edat)
<class 'str'>

Altres llenguatges, com Java, permeten declarar la variable sense inicialitzar-la. Per contra, a aquests llenguatges sovint hem d’indicar-li el tipus de dades que contindrà en el moment de la declaració i, un cop declarat el tipus, no el podrem canviar. Per exemple:

int edat;        // declaració de la variable edat
edat = 19;       // inicialització de la variable edat
edat = 20;       // modificació del valor de la variable edat
edat = "vintiu"; // (ERROR) això donaria un error de compilació. Estem intentant canviar el tipus!

De moment, però, només ens preocuparem per Python i deixarem Java per més endavant.
