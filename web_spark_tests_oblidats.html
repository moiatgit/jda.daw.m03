Hem acabat?¶

Què passarà si intentem tornar a passar els tests un altre cop? Doncs que, evidentment no passarà cap ni un! És clar, hem modificat composaSalutacio() sense canviar-los.

import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

public class TestMain{
    @Test
    public void quanLiPassoWendel() {
        String nom = "Wendel";
        String esperat = "Wendel";
        String obtingut = Main.composaSalutacio(nom);
        assertEquals(esperat, obtingut);
    }
    @Test
    public void quanLiPassoNull() {
        String nom = null;
        String esperat = "món";
        String obtingut = Main.composaSalutacio(nom);
        assertEquals(esperat, obtingut);
    }

    @Test
    public void quanLiPassoBuit() {
        String nom = "";
        String esperat = "món";
        String obtingut = Main.composaSalutacio(nom);
        assertEquals(esperat, obtingut);
    }
}

Si estiguéssim aplicant TDD (Test Driven Development) això no ens hagués passat doncs hauríem canviat els tests abans que el codi de composaSalutacio().
