#################
Diagrames de flux
#################

Hem vist com realitzar assignacions amb tres notacions diferents:
pseudocodi, Java i Python. Encara ens queden neurones per una quarta?

Existeix una notació gràfica per representar el flux d'instruccions
d'un programa: el *diagrama de flux*.

Per la representació de programes seqüencials, com els que sabem fer
fins ara, els diagrames de flux no són gaire espectaculars. Es tracta
d'una simple caixa envoltant les instruccions. La situació millorarà
una mica quan mirem les instruccions d'entrada/sortida.

pugui ajudar a entendre el funcionament del codi.
