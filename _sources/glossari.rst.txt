##################
Glossari de Python
##################

En aquesta pàgina trobaràs un recull de diferents elements de Python que
apareixen als continguts d'aquest curs.

.. glossary::
    :sorted:

    type()
        La funció ``type()`` ens permet saber quin és el tipus d'un
        element.

        .. code-block:: python

            >>> type(8)
            <class 'int'>
