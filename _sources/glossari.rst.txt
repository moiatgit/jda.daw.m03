##################
Glossari de Python
##################

En aquesta pàgina trobaràs un recull de diferents elements de Python que
apareixen als continguts d'aquest curs.

.. glossary::
    :sorted:

    int()
        La funció ``int()`` converteix a enter un valor.

        .. code-block:: python

            >>> int('123')
            123
            >>> int(123.456)
            123

    len()
        La funció ``len()`` ens permet saber el nombre d'elements que té un objecte com ara un
        *string*.

        .. code-block:: python

            >>> len("Hola")
            4

    type()
        La funció ``type()`` ens permet saber quin és el tipus d'un
        element.

        .. code-block:: python

            >>> type(8)
            <class 'int'>

    str.islower()
        La funció ``islower()`` del tipus string, ens permet saber si aquest correspon lletres en minúscules.

        .. code-block:: python

            >>> "h".islower()
            True
            >>> "1".islower()
            False

    str.isupper()
        La funció ``isupper()`` del tipus string, ens permet saber si aquest correspon lletres en minúscules.

        .. code-block:: python

            >>> "h".isupper()
            False

    str.isalpha()
        La funció ``isalpha()`` del tipus string, ens permet saber si aquest correspon a lletres.

        .. code-block:: python

            >>> "!".isalpha()
            False
            >>> "1".isalpha()
            False
            >>> "H".isalpha()
            True
