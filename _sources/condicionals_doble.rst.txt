###################
L'alternativa doble
###################

"Si plou llavors treu el paraigües, sinó deixa'l a casa"

Podem codificar aquesta condició de la següent manera:

.. code-block:: python

    if plou:
        treu_el_paraigues()

    if not plou:
        deixa_paraigues_a_casa()

Si bé això funciona, resulta una mica repetitiu. Molts llenguatges, com ara el
Python, ofereixen la possibilitat d'expressar aquest *sinó*.

En diagrama de flux ho representaríem com:

.. graphviz::
    :align: center

    digraph {
        inici[shape="box", style=rounded];
        final[shape="box", style=rounded];
        plou[shape="diamond", style=""];
        "treu el paraigües"[shape="rectangle", style=""]
        "deixa'l a casa"[shape="rectangle", style=""]

        inici-> plou;
        plou -> "treu el paraigües"[label="cert"];
        plou -> "deixa'l a casa"[label="fals"];
        "treu el paraigües" -> final;
        "deixa'l a casa" -> final;
    }

Fixa't com, amb aquesta nova forma de condicional, podem especificar
quines instruccions a executar quan s'acompleix la condició i quines quan no es
compleix.

En Python:

.. code-block:: python

    if plou:
       treu_el_paraigues()
    else:
       deixa_paraigues_a_casa()

Exercici 3. ☞ Dos en ordre
==========================

Desenvolupa un programa que demani dos números i escrigui els dos números
ordenats creixentment (de menor a major). Per exemple, si els números són
3 i 2, el resultat serà 2 3

Indica amb un comentari, que passa si els dos números són iguals.

Exemple d'interacció: ::

    $ python3 dosenordre.py
    Primer?
    3
    Segon?
    2
    2 i 3


Exercici 4. ☼ Quadrat dretà o esquerrà
======================================

Desenvolupa un programa que demani als usuaris un color, un sentit (dreta o
esquerra) i una longitut de costat.

Un cop obtingudes totes les dades, el programa crearà una tortuga que dibuixarà
el quadrat amb el color i dimensions indicats, i en el sentit demanat.

La simulació del programa podria ser similar a:

.. code-block:: none

    $ python3 quadratsentit.py
    Color?
    red
    Sentit (dreta o esquerra)?
    dreta
    Longitut costat?
    100

El resultat serà una finestra amb un dibuix d'un quadrat dibuixat amb girs cap a
la dreta.
