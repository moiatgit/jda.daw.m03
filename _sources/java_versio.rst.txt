###############
Versió del Java
###############

El nom *java* és tant la manera com la que ens referim al llenguatge de
programació, com l'especificació d'aquest llenguatge (entre d'altres
coses). És una idea molt semblant a la de SQL, una especificació i
diferents implementacions.

Durant el curs farem servir la implementació de Java anomenada *OpenJDK*
en la seva versió 11. Si tens qualsevol altra implementació o/i versió de
java ja instal·lada, sempre que la versió sigui >=1.8) no hauries de tenir
cap problema per seguir aquesta part del curs.

Comprova la teva versió amb les comandes:

.. code-block:: sh

    $ javac --version
    javac 11.0.5
    $ java --version
    openjdk 11.0.5 2019-10-15
    OpenJDK Runtime Environment (build 11.0.5+10-post-Debian-1bpo91)
    OpenJDK 64-Bit Server VM (build 11.0.5+10-post-Debian-1bpo91, mixed mode, sharing)

Si no tens cap instal·lada, fes (o demana fer si no tens privilegis
d'administració):

.. code-block:: sh

    $ sudo apt install openjdk-11-jre openjdk-11-jdk

.. note:: Et recomano que no t'obsessionis gaire amb la versió de Java. La
    darrera política dels seus propietaris (Oracle) és la d'oferir
    actualitzacions freqüents que sovint incorporen només petits canvis
    respecte l'anterior.

    Que tampoc no et confongui el nom de la versió. Malgrat la versió que fem
    servir aquest curs és la 1.11, a molts llocs sentiràs parlar de *Java 11*.
    La raó és històrica i té a veure amb polítiques de màrqueting que no
    t'haurien de costar gaire trobar si tens gust per converses de safareig.
