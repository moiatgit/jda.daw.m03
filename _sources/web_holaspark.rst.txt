##########
Hola Spark
##########

Com hem dit anteriorment, en aquesta part del curs farem servir el
*microframework* `Spark <http://sparkjava.com/>`_

A continuació t'explicaré com fer la instal·lació i configuració fins a
aconseguir crear una aplicació Web amb Spark.

Ho descobrirem passet a passet, amb dos exemples molt bàsics:

* una aplicació Web que mostra salutacions

  Aquest serà el que t'aniré oferint jo

* una aplicació Web que manegarà paraules capicua

  Aquest l'hauràs d'anar desenvolupant tu a partir del que aniràs
  descobrint a cada apartat.
