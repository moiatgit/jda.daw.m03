#####################################
Conversió entre sistemes de numeració
#####################################

Per passar d’una base **a** a una base **b** es passa de **a** a base decimal i d’aquesta a base **b**.

Per passar d’una base b a base 10 es desenvolupa el polinomi equivalent.

Ex. El número 32 en octal (base 8) es pot representar com 3*8^1+2*8^0=24+2=26 en base decimal.

Ex. El número F3A en hexadecimal (base 16) es pot representar com F*16^2+3*16^1+A*16^0=15*16^2+3*16+10 = 15*256+3*16+10 = 3898

Per passar de base decimal a una base b cal realitzar divisions successives per la base destinació sobre el quocient, fins que aquest sigui menor que la base

La conversió amb decimals d’una base a a base decimal es realitza considerant el polinomi equivalent
amb exponents negatius.

La conversió amb decimals de base decimal a una base b es realitza amb multiplicacions successives
quedant-nos amb la part entera.

Quan la conversió es realitza entre dos bases que són potència una de l’altra, llavors es pot fer
sense passar per base decimal.

Ex. El número 1100100101 en binari es passa a octal agrupant de tres en tres 001 100 100 101 i
convertint directament a octal: 1445

Ex. El número F3A en hexadecimal es passa a binari traduint directament cada dígit en grups de
quatre: 1111 0011 1010
