##########
Seqüències
##########

Hem presentat les :doc:`seqüències <sequencies>` com una agrupació de dades
ordenades, de manera que podem recórrer-les de la primera a la darrera.

Amb Python vam fer servir les tuples, que ens permetien una manera molt còmoda
d'implementar seqüències, doncs estaven integrades dins del llenguatge.

Java disposa també d'un tipus de seqüència integrat dins el llenguatge: els
*arrays*. 

Els *arrays* són una manera molt eficient i còmoda de guardar informació
seqüèncial en Java, i per això els estudiarem en primer lloc. Els *arrayus* però
ens presentaran algunes dificultats a l'hora d'ampliar-los. Això ens portarà a
l'ús d'una estructura una mica més complexa anomenada ``List`` que ens
solventarà bona part de les limitacions dels *arrays* pagant un lleuger preu en
comoditat d'ús.
