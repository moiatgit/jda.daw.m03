################
Ritual Hola Món!
################

Molt bé, ja tenim l'entorn per guardar les nostres respostes. Quan comencem a
programar? Doncs ara mateix!

Segueix les passes:

1. Obre un terminal (ex. <ctrl><alt>t)

2. Escriu a la línia de comandes (no incloguis ``$``)

   .. code-block:: bash

       $ python3

3. Escriu després de ``>>>`` (no incloguis tampoc ``>>>``)

   .. code-block:: python

       >>> print("Hola Món!")

4. Si has vist la salutació al món, felicitats pel teu primer programa!

5. Si apareix algun error, revisa que el text que has introduït
   contingui **exactament** l'indicat. Corregeix el que calgui
   i torna a executar fins que no s'indiqui cap error.

6. Surt de l'intèrpret de Python prement ``ctrl-d``

Ara ja som programadors iniciats. Tenim, però, un petit problema: el nostre programa l'hem de tornar
a escriure cada cop que el vulguem executar. Una solució és guardar el codi del nostre programa (sí,
allò que comença amb ``print``) en un fitxer de text.

Segueix novament les instruccions:

1. Escriu a la consola:

   .. code-block:: bash

    $ mkdir -p ~/programacio/exemple.holamon
    $ cd ~/programacio/exemple.holamon
    $ gedit holamon.py

2. A la finestra que t'apareixerà, escriu la línia amb el codi:

   .. code-block:: python

        print("Hola Món!")

   Guarda el contingut del fitxer, per exemple amb ``ctrl-s`` i surt de gedit ``ctrl-q``

3. Novament a la consola, comprovarem que tenim el nou fitxer:

   .. code-block:: bash

        $ ls
        holamon.py
        $ cat holamon.py
        print("Hola Món!")

4. Executem el nostre programa des de la consola:

   .. code-block:: bash

        $ python3 holamon.py
        Hola Món!

5. Si has vist la salutació al món, felicitats pel teu segon programa!

6. si apareix algun error, revisa que hagis seguit les passes
   **exactament**. Torna a realitzar totes les passes tot corregint el que calgui, fins que no
   trobis cap error.

Ja tenim el nostre programa de manera que el podem tornar a executar tants cops com vulguem.

Encara ens podem estalviar haver d'escriure ``python3`` per executar el nostre programa.
Per fer-ho, segueix les passes:

1. Edita el fitxer ``holamon.py`` i fes que contingui el següent:

   .. code-block:: python

        #! /usr/bin/env python3
        print("Hola Món!")

   Quan la primera línia d'un fitxer comença amb ``#!``, l'intèrpret de Bash entén que aquest és el
   programa que ha de processar la resta del contingut del fitxer. Es coneix com a *shebang*.

2. Ara, cal dir-li al sistema operatiu que aquest fitxer és executable. Des de línia de comandes,
   escrivim:

   .. code-block:: none

        $ ls -l holamon.py
        -rw-r--r-- 1 profe profe 44 set  3 11:03 holamon.py
        $ chmod u+x holamon.py
        $ ls
        -rwxr--r-- 1 profe profe 44 set  3 11:03 holamon.py

   Fixa't com ara el fitxer conté la marca ``x`` als permisos

3. Finalment executem el nostre programa:

   .. code-block:: bash

        $ ./holamon.py
        Hola Món!


Exercici 3. Presentem-nos
=========================

En aquesta secció et proposo una senzilla modificació respecte el que hem vist
al programa ``holamon.py``. La seva realització et permetrà entendre una mica
més com es fa servir l'eina de control de versions ``git``.

Segueix les instruccions:

#. Ves a la carpeta de treball del teu PC i actualitza canvis que puguin estar
   al repositori remot

   .. code-block:: bash

        $ cd ~/introprg
        $ git pull origin master

   En cas que et doni algun error, és possible que el teu espai de treball
   tingués canvis que no has registrat. Fes ``git status`` perquè ``git`` et
   recordi quins canvis vas fer, i fes el ``commit`` oportú. En cas de dubtes,
   avisa el teu docent que el tens desitjant ajudar-te.

#. Crea la carpeta ``00_03_presentemnos``

   .. code-block:: bash

        $ mkdir 00_03_presentemnos
        $ cd 00_03_presentemnos

#. Crea un nou fitxer ``empresento.py`` i escriu-hi codi tal que, en ser
   executat escrigui per pantalla el mateix que ``holamon.py``, excepte que, en
   comptes de *Hola Món!*, sortirà un missatge similar al següent:

   .. code-block:: bash

       Hola, sóc la Adelita, la futura millor desenvolupadora del Món!

   Per favor, si no et dius *Adelita*, canvia el nom. Pots modificar també la resta
   de la frase amb quelcom que sentis còmode.

#. Aconsegueix que el programa pugui ser executat de la següent manera:

   .. code-block:: bash

      $ ./empresento.py

#. Afegeix els canvis al control de versions.

   Pots comprovar que ``git`` t'informa que hi ha un nou fitxer sense registrar,
   tot fent ``git status``.

   Com ara hem creat un nou fitxer, ens caldrà informar a ``git`` que el volem
   gestionar:

   .. code-block:: bash

        $ git add empresento.py

   Ara que el fitxer ja és conegut per ``git``, demanem que afegeixi els canvis:

   .. code-block:: bash

        $ git commit -am "Completat l'exercici 3"

#. Finalment pugem els canvis a GitHub:

   .. code-block:: bash

        $ git push origin master

   Com sempre, indica el teu password


A banda de l'exercici, et recomano que tu mateix et defineixis canvis que
t'agradaria provar. Crea d'altres fitxers pels teus experiments, que pots
mantenir sense problemes a aquesta carpeta.  La màxima és: **sense por**. No
pots trencar res!
