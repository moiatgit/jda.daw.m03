Considera el següent programa:

.. code-block:: python
    :emphasize-lines: 5
    :linenos:

    """ Programa que va demanant nombres enters fins que li diuen prou.
        En acabar, mostra quants s'han introduït correctament """
    quants = 0
    entrada = ''
    while entrada != 'prou':
        entrada = input('Introdueix un nombre o "prou" per finalitzar:\n')
        if entrada.isdigit():
            quants += 1
        elif entrada != 'prou':
            print('No és un nombre enter vàlid')
    print('Valor enters introduïts: %s' % quants)
