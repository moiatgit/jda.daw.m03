#############################
Configurem la màquina virtual
#############################

Ara ja tenim iniciada la màquina virtual. La seva configuració es troba al
fitxer ``Vagrantfile``.

L'obrirem amb el nostre editor preferit i localitzarem la secció següent:

.. code-block:: none
    :emphasize-lines: 3

      # Create a private network, which allows host-only access to the machine
      # using a specific IP.
      # config.vm.network "private_network", ip: "192.168.33.10"

Editarem la línia remarcada, de manera que es llegeixi:

.. code-block:: none
    :emphasize-lines: 3

      # Create a private network, which allows host-only access to the machine
      # using a specific IP.
      config.vm.network "private_network", ip: "192.168.33.10"

Amb això estem indicant que la màquina virtual disposarà una interfície de
xarxa amb la IP ``192.168.33.10``.

Aquest port és el que fa servir per defecte el servidor de *PostgreSQL*.
Per simplicitat, et recomano que mantinguis aquests valors.
